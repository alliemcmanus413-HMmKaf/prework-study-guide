HTML

  • Use "<>" for opening tag; use "< />" for closing tag
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Only use h1 once, use h2-h6 for smaller headers on the rest of webpage.
  • The title is the element shown in the tab in the browser.

CSS

  • CSS describes how HTML elements will appear on webpage.
  • CSS is written in markup language.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is used to share repositories and make working on code collaboratively possible.
  • Git uses SSH keys to confirm the connection between Git users and personal workspaces.

JavaScript

  • JavaScript is a lightweight programming language used to provide logic and interactivity for websites, as described in the PreWork.
  • JavaScript can create effects in web browsers.
  • JavaScript is one of the most popular programming languages.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
